# Alpha Vantage (alpha-vantage)

Alpha Vantage provides a single REST market-data API at `https://www.alphavantage.co/query` exposing 100+ functions across stocks, fundamentals, forex, crypto, commodities, economic indicators, technical indicators (50+), Alpha Intelligence (news, sentiment, insider transactions), and options data.
